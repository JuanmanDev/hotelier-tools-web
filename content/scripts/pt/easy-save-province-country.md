## O que este script faz?

Este script simplifica o processo de preenchimento dos detalhes do hóspede, fornecendo uma lista de preenchimento automático para as províncias espanholas e preenchendo automaticamente o país.

**Recursos:**
- Mostra uma lista de províncias espanholas para seleção ao focar no campo de província.
- Define automaticamente o país para "Spain" quando uma província é selecionada.
- Adiciona um sublinhado `_` para os campos de nome e sobrenome vazios, facilitando a próxima integração com o INE.
