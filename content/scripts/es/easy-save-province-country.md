## ¿Qué hace este script?

Este script agiliza el rellenado de los detalles del huésped, proporcionando una lista autocompletable para las provincias españolas y rellenando automáticamente el país.

**Características:**
- Muestra una lista de provincias españolas para seleccionar al hacer foco en el campo de provincia.
- Establece automáticamente el país como "Spain" al seleccionar una provincia.
- Rellena con un guion bajo `_` los campos de nombre y apellido si están vacíos, facilitando la próxima integración con el INE.
